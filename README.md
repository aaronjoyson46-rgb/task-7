Overview:
This project applies statistical techniques to a synthetic business dataset. It covers descriptive statistics, distribution analysis, correlation, hypothesis testing, and confidence interval estimation in a business context.

Files Included:
- Week7_Statistical_Analysis.ipynb  : Main analysis notebook
- week7_statistical_dataset.csv      : Synthetic dataset with 300 rows
- week7_stats_summary.txt            : Summary generated by the notebook (created after running all cells)

Techniques Used:
- Descriptive statistics (mean, median, standard deviation)
- Histograms and distribution analysis
- Pearson correlation between marketing spend and revenue
- Independent t-test for Online vs Offline sales
- 95 percent confidence interval for mean sales

How to Run:
1. Install the required libraries:
   pip install pandas numpy seaborn matplotlib scipy

2. Place the notebook and CSV file in the same folder.

3. Open the notebook:
   jupyter notebook Week7_Statistical_Analysis.ipynb

4. Run all cells in order to reproduce the analysis and generate week7_stats_summary.txt.
